## About

I am a full-stack developer who builds complete web applications end to end from database schema and backend services through to the interfaces users actually touch. I work primarily with Laravel, Python, and React, and I care about the parts of engineering that determine whether software survives its second year: clear structure, predictable performance, and code that the next developer can reason about.

What defines my current practice is that AI is integrated into both what I build and how I build it. On the product side, I embed intelligence directly into applications — LLM-powered features, AI-assisted data processing, automated classification and extraction — treating model inference as a first-class dependency with proper timeout, retry, and fallback handling rather than a bolted-on demo. On the workflow side, I build with AI-native tooling: Claude Code for accelerated development and refactoring, n8n for orchestrating automation across services, and Selenium-driven pipelines for large-scale data extraction. The result is that I ship more surface area, faster, without trading away architectural discipline.

I approach every project with a product engineering mindset  understanding the problem before choosing the framework, designing for the load the system will realistically see, and treating maintainability as a feature rather than an afterthought.
